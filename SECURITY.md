# Security Policy

## Reporting a Vulnerability

This is a cryptographic standard specification. If you find a security issue:

1. **DO NOT** open a public issue
2. Email: security@kairoslab.xyz
3. Or use the GitHub Security Advisory feature (Security tab → Report a vulnerability)

We will respond within 48 hours.

## Scope

- Interface design vulnerabilities
- Proof system specification gaps
- Cryptographic primitive misuse
- Privacy leakage vectors in the standard itself

## Out of Scope

- Implementation bugs (reference implementation not yet published)
- Gas optimization suggestions
- Feature requests
